CRS
===

Package for working with 
SPARSE matrices in CRS (Compressed Row Storage) format. 
===
VERY USEFUL for students.

OPERATIONS:
===

1) Initialization 
2) +, x, =
3) Transposition
4) I/O from/to file
5) Conversion (DENSE <-> CRS)
